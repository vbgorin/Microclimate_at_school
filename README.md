# Microclimate_at_school
Мы предлагаем решение прикладной задачи по мониторингу микроклимата в общеобразовательных учреждениях, основным показателем которого является качество воздуха. В работе использованы современных высокоточные датчики и сенсоры, а также передовые технологии анализа качества воздуха. Данная работа посвящена созданию системы датчиков непрерывного мониторинга микроклимата в общеобразовательном учреждении. Новизна работы заключается в использовании для оценки микроклимата таких параметров как: тепловой индекс , индекс качества воздуха, уровень запыленности, концентрация СО2, концентрация летучих органических веществ, освещенность.
